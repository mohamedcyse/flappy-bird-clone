#The game is a clone to the famous original virsion of the game "Flappy Bird" which
is a mobile game developed by Vietnamese video game artist and programmer Dong Nguyen. 


#LibGDX is used as an open source gaming library and cross platform.

#Adding sprites:

-Like textures for pipes and the bird from
http://androiddevcourse.com/flappybirdassets.zip.



#Collision detection 

-Shapes are used for the bird and the pipes (Circle and Rectangle)

-Intersectors are used to check for over laps

-BitmapFont is used to display the score.
